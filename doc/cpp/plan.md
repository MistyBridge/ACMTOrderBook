# C++ 撮合引擎重写计划

## 目标
将 Python py/ 目录下的订单簿重建引擎重写为 C++，保持算法完全一致，同时做模块化拆分。

---

| 版本 | 状态 | 说明 |
|------|------|------|
| CPP v1 | ✅ 已完成 | 单线程版本，基础 C++ 重写 |
| CPP v2 | ✅ 已完成 | 多线程优化版本（生产者-消费者架构） |
| CPP v2.1 | 🚧 进行中 | 性能极致优化（五批次计划） |

## CPP v2：多线程优化版本

### v2 核心优化
| 优化项 | 说明 | 状态 |
|--------|------|------|
| SPSC 无锁队列 | 环形缓冲区 + atomic 读写指针 | ✅ 已完成 |
| 内存池 | 侵入式空闲链表，2x 增长 | ✅ 已完成 |
| 缓存行对齐 | alignas(64) 防止 false sharing | ✅ 已完成 |
| 批量处理 | pop_batch 一次取 64 条 | ✅ 已完成 |
| CPU 亲和性 | 生产者/消费者绑定不同物理核 | ✅ 已完成 |
| 延迟统计 | 环形缓冲区 + nth_element | ✅ 已完成 |
| 字段重排 | ObOrder 40B → 32B | ✅ 已完成 |
| 分支预测 | LIKELY/UNLIKELY 宏 | ✅ 已完成 |
| 编译优化 | -O2 -march=native | ✅ 已完成 |

### 性能指标
- 吞吐量：95,455 msg/s（+47.7% vs v1）
- 延迟：p50=0.3μs, p99=71.5μs, p99.9=521.8μs, pmax=957.3μs
- 编译选项：-O2 -march=native

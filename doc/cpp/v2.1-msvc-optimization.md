# v2.1 MSVC 编译优化报告

## 📊 优化概述

**优化目标**：使用 MSVC 2022 编译器优化，提升单线程性能。

**优化方法**：
1. MSVC 2022 /O2 /GL /arch:AVX2 /LTCG 编译选项
2. HybridLevelBook 数据结构（排序数组 + std::map 回退）
3. 分支预测宏（LIKELY/UNLIKELY）
4. ObOrder 字段重排（40B → 32B）

**优化效果**：吞吐量从 95K msg/s 提升至 212K msg/s，**提升 123%**。

---

## 🔧 技术实现

### 1. MSVC 编译优化

**编译选项**：
```bash
/O2      # 最大速度优化
/GL      # 全程序优化
/arch:AVX2  # AVX2 指令集
/LTCG    # 链接时代码生成
```

**效果**：
- 指令级并行优化
- 循环展开
- 内联函数优化
- SIMD 指令优化

### 2. HybridLevelBook 数据结构

**设计**：
- 小规模（n≤256）：排序数组，二分查找
- 大规模（n>256）：std::map 回退

**优势**：
- 缓存友好（连续内存）
- 分支预测友好
- 内存占用小

### 3. 分支预测宏

```cpp
#define LIKELY(x)   __builtin_expect(!!(x), 1)
#define UNLIKELY(x) __builtin_expect(!!(x), 0)
```

**效果**：
- 热路径优化
- 减少分支预测失败

### 4. ObOrder 字段重排

**优化前**：40B（有 padding）
**优化后**：32B（无 padding）

**效果**：
- 减少内存占用
- 提高缓存命中率
